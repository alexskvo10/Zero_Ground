# Тестирование типов стен

## Что было сделано

Добавлены два типа стен:
1. **Бетонные стены** (серые, цвет: RGB 150, 150, 150) - генерируются с вероятностью 70%
2. **Деревянные стены** (коричневые, цвет: RGB 139, 90, 43) - генерируются с вероятностью 30%

## Изменения в коде

### Сервер (Zero_Ground/Zero_Ground.cpp):
- Добавлен enum `WallType` с тремя значениями: None, Concrete, Wood
- Структура `Cell` теперь хранит тип стены вместо bool
- Функция `generateMap()` генерирует стены с заданными вероятностями
- Функция `renderVisibleWalls()` отрисовывает стены разными цветами
- Все проверки коллизий обновлены для работы с типами стен

### Клиент (Zero_Ground_client/Zero_Ground_client.cpp):
- Аналогичные изменения для синхронизации с сервером

## Как протестировать

1. Запустите сервер: `Zero_Ground\x64\Debug\Zero_Ground.exe`
2. Запустите клиент: `Zero_Ground_client\x64\Debug\Zero_Ground_client.exe`
3. Подключитесь к серверу (localhost)
4. Нажмите Ready на обоих окнах

## Что проверить

✓ Стены генерируются двух цветов (серые и коричневые)
✓ Примерно 70% стен серые (бетонные), 30% коричневые (деревянные)
✓ Коллизии работают для обоих типов стен
✓ Игроки не могут проходить сквозь стены любого типа
✓ Fog of War корректно применяется к обоим типам стен
✓ Производительность не ухудшилась

## Статистика генерации

При запуске сервер выводит статистику:
```
Generated X walls (Concrete: Y, Wood: Z)
```

Проверьте, что соотношение примерно 70/30.
