# Инструкция по тестированию всплывающих подсказок оружия

## Подготовка к тестированию

### 1. Компиляция клиента
Убедитесь, что у вас настроены пути к SFML библиотеке, затем скомпилируйте клиент:
```bash
compile_client.bat
```

### 2. Запуск сервера
Запустите сервер для тестирования:
```bash
Zero_Ground\Debug\Zero_Ground.exe
```

### 3. Запуск клиента
Запустите клиент:
```bash
Zero_Ground_client\Debug\Zero_Ground_client.exe
```

## Тестовые сценарии

### Сценарий 1: Базовая функциональность tooltip
1. Подключитесь к серверу (127.0.0.1)
2. Нажмите "READY" и дождитесь начала игры
3. Подойдите к любому магазину (красный квадрат 20×20 пикселей)
4. Нажмите клавишу **B** для открытия магазина
5. Наведите курсор мыши на любой слот оружия

**Ожидаемый результат:**
- Появляется всплывающее окно 320×290 пикселей
- Окно имеет золотую рамку
- Отображается название оружия золотым цветом
- Показана цена светло-зелёным цветом
- Видны все 8 характеристик оружия

### Сценарий 2: Позиционирование tooltip
1. Откройте магазин (клавиша **B**)
2. Наведите курсор на оружие в **левом верхнем углу** списка

**Ожидаемый результат:**
- Tooltip появляется справа и снизу от курсора

3. Наведите курсор на оружие в **правом нижнем углу** списка

**Ожидаемый результат:**
- Tooltip автоматически смещается влево и/или вверх
- Tooltip полностью виден на экране

### Сценарий 4: Взаимодействие с другими элементами UI
1. Откройте магазин
2. Наведите курсор на оружие (появится tooltip)
3. Переместите курсор на другое оружие

**Ожидаемый результат:**
- Tooltip мгновенно переключается на новое оружие
- Нет задержек или мерцания

4. Уберите курсор со всех слотов оружия

**Ожидаемый результат:**
- Tooltip исчезает

### Сценарий 5: Tooltip при покупке
1. Откройте магазин
2. Наведите курсор на оружие (появится tooltip)
3. Кликните на слот оружия для покупки

**Ожидаемый результат:**
- Оружие покупается (если достаточно денег и есть место)
- Tooltip остаётся видимым, если курсор всё ещё на слоте
- Информация в tooltip актуальна

### Сценарий 6: Tooltip в разных разрешениях экрана
1. Переключитесь в оконный режим (клавиша **Esc**)
2. Откройте магазин
3. Наведите курсор на разные слоты оружия

**Ожидаемый результат:**
- Tooltip корректно отображается в оконном режиме
- Позиционирование работает правильно

4. Переключитесь обратно в полноэкранный режим (клавиша **Esc**)
5. Повторите тест

**Ожидаемый результат:**
- Tooltip корректно отображается в полноэкранном режиме

## Проверка визуального качества

### Цвета
- Фон tooltip: Тёмно-серый, почти чёрный
- Рамка: Золотая (яркая)
- Название оружия: Золотое, жирное
- Цена: Светло-зелёная
- Названия характеристик: Светло-серые
- Значения характеристик: Белые, жирные

### Читаемость
- Весь текст должен быть чётким и читаемым
- Нет наложений текста
- Разделительная линия видна между ценой и характеристиками

### Анимация
- Tooltip появляется мгновенно при наведении
- Tooltip исчезает мгновенно при уходе курсора
- Нет задержек или лагов

## Известные ограничения
- Tooltip отображается только в UI магазина
- Tooltip не сохраняется при закрытии магазина
- Tooltip следует за курсором только при смене слота оружия
