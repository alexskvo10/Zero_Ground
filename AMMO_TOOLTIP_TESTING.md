# Руководство по тестированию подсказок патронов

## Быстрый старт

### Компиляция
```batch
compile_client.bat
```

### Запуск
1. Запустите сервер: `Zero_Ground.exe`
2. Запустите клиент: `Zero_Ground_client.exe`
3. Подключитесь к серверу (127.0.0.1)

## Тестовые сценарии

### Тест 1: Базовое отображение подсказки
**Цель:** Проверить, что подсказка появляется при наведении

**Шаги:**
1. Откройте магазин клавишей **B** (рядом с магазином)
2. Найдите колонку "Ammo" (4-я колонка справа)
3. Наведите курсор на слот "Bullets 9x18"

**Ожидаемый результат:**
- Появляется всплывающая подсказка
- Отображается название: "Bullets 9x18"
- Показана цена: "$100"
- Указано количество: "10 rounds"
- Перечислены совместимые пистолеты

### Тест 2: Информация о совместимости
**Цель:** Проверить корректность информации о совместимых типах оружия

**Шаги для каждого типа патронов:**

#### Bullets 9x18
- Наведите на слот "Bullets 9x18"
- Проверьте список: USP, Glock-18, Five-SeveN, R8 Revolver

#### Bullets 5,45x39
- Наведите на слот "Bullets 5,45x39"
- Проверьте список: Galil AR, M4, AK-47

#### Bullets 7,62x54
- Наведите на слот "Bullets 7,62x54"
- Проверьте список: M10, AWP, M40

**Ожидаемый результат:**
- Для каждого типа патронов показаны правильные совместимые типы оружия
- Список отображается голубым цветом
- Форматирование читаемое и понятное

### Тест 3: Позиционирование подсказки
**Цель:** Проверить адаптивное позиционирование

**Шаги:**
1. Наведите на верхний слот патронов (курсор в центре экрана)
   - Подсказка должна появиться справа-снизу от курсора
   
2. Наведите на слот, когда курсор у правого края экрана
   - Подсказка должна появиться слева от курсора
   
3. Наведите на слот, когда курсор у нижнего края
   - Подсказка должна появиться сверху от курсора

**Ожидаемый результат:**
- Подсказка всегда остаётся в пределах экрана
- Не перекрывает нижние UI элементы (150px резерв)
- Минимальный отступ от краёв: 10px

### Тест 4: Переключение между слотами
**Цель:** Проверить корректную смену подсказок

**Шаги:**
1. Наведите на "Bullets 9x18"
2. Переместите курсор на "Bullets 5,45x39"
3. Переместите курсор на "Bullets 7,62x54"
4. Уберите курсор со всех слотов

**Ожидаемый результат:**
- При переходе между слотами подсказка мгновенно меняется
- Нет задержек или мерцания
- При уходе курсора подсказка исчезает
- Нет утечек памяти (старые объекты удаляются)

### Тест 5: Взаимодействие с подсказками оружия
**Цель:** Проверить совместную работу подсказок оружия и патронов

**Шаги:**
1. Наведите на слот оружия (например, USP)
   - Должна появиться подсказка оружия
2. Переместите курсор на слот патронов (Bullets 9x18)
   - Подсказка оружия должна исчезнуть
   - Должна появиться подсказка патронов
3. Переместите обратно на оружие
   - Подсказка патронов исчезает
   - Появляется подсказка оружия

**Ожидаемый результат:**
- Одновременно отображается только одна подсказка
- Переключение происходит плавно
- Нет конфликтов между подсказками

### Тест 6: Визуальное оформление
**Цель:** Проверить соответствие дизайну

**Проверяемые элементы:**
- ✅ Золотая рамка (2px)
- ✅ Тёмно-серый фон с прозрачностью
- ✅ Название золотым цветом, жирный шрифт
- ✅ Цена зелёным цветом
- ✅ Список оружия голубым цветом
- ✅ Разделительная линия серого цвета
- ✅ Читаемые размеры шрифтов

### Тест 7: Производительность
**Цель:** Убедиться в отсутствии проблем с производительностью

**Шаги:**
1. Быстро перемещайте курсор между всеми слотами патронов
2. Повторите 20-30 раз
3. Следите за FPS и плавностью

**Ожидаемый результат:**
- FPS остаётся стабильным (60 FPS)
- Нет задержек при отрисовке
- Нет утечек памяти
- Плавная работа UI

## Проверка на разных разрешениях

### Fullscreen (1920x1080)
- Подсказки корректно позиционируются
- Текст читаемый
- Не выходят за границы экрана

### Windowed (800x600)
- Подсказки адаптируются к меньшему размеру
- Остаются в пределах окна
- Текст остаётся читаемым

## Известные ограничения

1. Подсказка не отображается, если магазин закрыт
2. Подсказка исчезает при закрытии магазина
3. Одновременно может отображаться только одна подсказка (оружие ИЛИ патроны)

## Отчёт об ошибках

При обнаружении проблем укажите:
- Тип патронов
- Позицию курсора
- Разрешение экрана
- Скриншот (если возможно)
- Шаги для воспроизведения
