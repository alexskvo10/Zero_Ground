# Главное меню клиента

## Описание

При запуске клиента теперь отображается главное меню с фоновым изображением и тремя кнопками в левом нижнем углу.

## Структура меню

### Фоновое изображение
- Файл: `Fon.png`
- Отображается на весь экран (масштабируется автоматически)

### Кнопки меню (левый нижний угол)

1. **Play** (зеленая кнопка)
   - При нажатии переходит к экрану ввода IP-адреса сервера
   - Фон экрана ввода IP также использует `Fon.png`

2. **Developers** (серая кнопка)
   - Пока не активна (будет реализована позже)

3. **Hints** (серая кнопка)
   - Пока не активна (будет реализована позже)

## Последовательность экранов

1. **MenuScreen** - Главное меню с кнопками (с фоном Fon.png)
2. **ConnectScreen** - Экран ввода IP-адреса сервера (с фоном Fon.png)
3. **ErrorScreen** - Экран ошибки подключения (с фоном Fon.png)
4. **Connected** - Экран успешного подключения с кнопкой READY (с фоном Fon.png)
5. **WaitingForStart** - Ожидание начала игры (с фоном Fon.png)
6. **ConnectionLost** - Экран потери соединения (с фоном Fon.png)
7. **MainScreen** - Игровой экран

## Технические детали

- Состояния клиента: `MenuScreen`, `ConnectScreen`, `Connected`, `WaitingForStart`, `MainScreen`, `ErrorScreen`, `ConnectionLost`
- Начальное состояние: `MenuScreen`
- Размер кнопок: 300x60 пикселей
- Отступ от края: 50 пикселей
- Расстояние между кнопками: 20 пикселей

## Управление

- **ESC** - переключение полноэкранного режима (работает на всех экранах)
- **Левая кнопка мыши** - нажатие на кнопки меню
- **Ввод текста** - ввод IP-адреса на экране ConnectScreen
- **Enter** - подключение к серверу (на экране ConnectScreen)
