# Примечание по компиляции клиента

## Проблема
Проект настроен на компиляцию для платформы x64, но текущие библиотеки SFML в `D:\SFML-2.6.1\lib` являются 32-битными (x86).

## Решение
Для успешной компиляции клиента необходимо:

1. **Скачать 64-битную версию SFML 2.6.1**
   - Перейдите на https://www.sfml-dev.org/download/sfml/2.6.1/
   - Скачайте версию для Visual C++ 17 (2022) - 64-bit

2. **Установить библиотеки**
   - Распакуйте архив в `D:\SFML-2.6.1-x64\` (или другую папку)
   - Обновите пути в файле `Zero_Ground_client.vcxproj`:
     - Найдите строки с `D:\SFML-2.6.1\include` и `D:\SFML-2.6.1\lib`
     - Замените на пути к 64-битной версии

3. **Скомпилировать проект**
   ```
   msbuild Zero_Ground_client.sln /p:Configuration=Debug /p:Platform=x64
   ```

## Альтернатива
Если у вас уже есть 64-битные библиотеки SFML в другой папке, просто обновите пути в `Zero_Ground_client.vcxproj`.
